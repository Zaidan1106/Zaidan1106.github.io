About Me

Sedikit perkenalan tentang saya.

Nama saya Muhammad Zaidan Al Fath, biasa dipanggil Zaidan. Saya lahir di Jakarta, 11 Juni 2007. Saya adalah Anak ke-1 dari dua bersaudara. Saat ini saya sedang menempuh pendidikan di SMKN 9 KOTA BEKASI di mana saya mengambil jurusan SIJA dan saat ini saya menempati kelas XI SIJA 1. Kedepannya saya ingin berprofesi sebagai Web Developer.


Awal mula saya mengenal dunia coding ini bersumber dari sebuah komunitas belajar yang ada di SMKN 9 KOTA BEKASI ini, yaitu @meta.tech. yang saya ikuti sejak saya masih kelas X SIJA 1. Dari komunitas tersebutlah saya mulai mengenal apa itu coding web dan banyak hal tentang IT.
